#  Simple mdns browser

これは [mdns-js](https://github.com/mdns-js/node-mdns-js) を利用したmDNSサービスのブラウザです。

# 前提

Node.js で動きます。(Mac, Linux, Windows)

## 準備

モジュールをインストール

```
$ npm isntall
```

または


```
$ npm isntall mdns-js
```

# 使い方

## 全てのサービスをブラウズ


```
$ node browser.js
```

## サービス名を指定してブラウズ

```
$ node browser.js サービス名
```
